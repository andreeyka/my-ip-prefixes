# CLAUDE.md

## Project Overview

Генерация sing-box rule-set (.srs) с IP-префиксами, не принадлежащими странам СНГ.

Скрипт скачивает .srs файлы стран из репозитория [ipranges-singbox](https://github.com/you-oops-dev/ipranges-singbox),
извлекает IP-диапазоны, классифицирует всё IPv4-пространство многоуровневым алгоритмом
и генерирует компактный rule-set для маршрутизации не-СНГ трафика.

## Алгоритм

Многоуровневая классификация: IPv4 делится на блоки от крупных (/13) к мелким (/24).
На каждом уровне блок с покрытием СНГ-IP выше порога → "наш", 0% покрытия → "не наш",
остальные → дробятся на следующем уровне.

Дефолтные уровни: `13:1,17:2,20:5,24:20` → ~1600 префиксов, ~9% потерь, 0% утечки.

## Python Setup

- Python 3.13+
- Менеджер пакетов: uv
- Внешняя зависимость: sing-box CLI

```bash
uv sync
uv run main.py -o not-cis.json -c   # генерация not-cis.srs
uv run main.py --help                # все параметры
```

## Автообновление

GitHub Actions запускается каждые 4 часа, скачивает свежие данные,
генерирует not-cis.srs и коммитит при изменениях.
