## Исключённые провайдеры

IP этих провайдеров принудительно считаются зарубежными, даже если попадают в CIS-диапазоны.
Из CIS-пространства исключено **652,909 IP**.

| Провайдер | Префиксов в источнике |
|-----------|-----------------------|
| `google-all` | 18,029 |
| `cloudflare` | 15 |

## Использование в sing-box

```json
{
  "rule_set": [
    {
      "tag": "not-cis",
      "type": "remote",
      "url": "https://raw.githubusercontent.com/andreeyka/my-ip-prefixes/main/not-cis.srs",
      "format": "binary"
    }
  ]
}
```
